# Lexiwind

This is a simple React application where I utilize Lexiwind, a custom Tailwind CSS variant, for styling and shadcn/ui for UI components.

## Installation

1. npm i lexiwind

## Technologies Used

- React.js
- tailwind
- shadcn/ui
- lexical
